# Security Policy

## Supported Versions

This is a personal project. Only the latest version on the `master` branch is maintained.

## Reporting a Vulnerability

If you find a security issue — especially anything related to API key exposure or server-side vulnerabilities — please do **not** open a public issue.

Instead, reach out directly:

- **Email**: marxclethers@gmail.com
- **GitHub**: [@clethers](https://github.com/clethers)

I'll respond as soon as I can. For a personal project like this, there's no formal SLA, but I take it seriously.

## Notes

- The `.env` file is **not** committed to this repo. Never share your `GROQ_API_KEY` or any other keys publicly.
- If you see a leaked key in a commit, please flag it immediately so it can be rotated.
